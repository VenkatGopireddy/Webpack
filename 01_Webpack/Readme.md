# Webpack

- Installing Webpack
  - `https://webpack.github.io/`
  - `https://www.npmjs.com/`
    - Type `webpack` in the search bar will navigate to `https://www.npmjs.com/package/webpack`
    - Prerequisite: `Node JS` to be installed `https://nodejs.org/en/`
  - Creating Package.json file
    - `npm init`
